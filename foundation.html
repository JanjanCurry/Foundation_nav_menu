Horizontal Dropdown Menu

Set up: Dropdown menus build on the Menu component's syntax. Add the class .dropdown and the attribute data-dropdown-menu to the menu container to set up the dropdown.

Activate: To create dropdown menus, nest a new <ul> inside an <li>. You can nest further to create more levels of dropdowns.

Note that the <ul> goes after the <a>, and not inside of it.
